# 制造业销售数据分析项目

## 项目背景
为中小型冲压件制造企业搭建销售数据分析体系，识别产品趋势与风险，辅助业务决策。

## 核心成果

### 1. SQL分析体系（/sql/）
| 查询文件 | 用途 | 技术点 |
|:---|:---|:---|
| 24-25年度/月度销售额.sql | 基础业绩统计 | GROUP BY聚合 |
| top10产品.sql | 识别核心贡献产品 | ORDER BY+LIMIT |
| 不同产品月度销售额.sql | 产品维度透视 | 多字段分组 |
| 月度销售额环比变化.sql | 月度趋势分析 | 窗口函数LAG |
| 连续2个月销售额下滑预警.sql | 风险产品识别 | 窗口函数嵌套 |
| 24-25产品销售额年增长率.sql | 年度对比分析 | LAG+JOIN |

### 2. Python自动化分析（/python/）
| 文件 | 功能 | 技术点 |
|:---|:---|:---|
| 数据库连接与查询.ipynb | MySQL连接+pandas读取 | sqlalchemy+pymysql |
| 多产品趋势对比.ipynb | matplotlib可视化 | 折线图、多图对比 |
| 自动生成PDF报告.ipynb | 数据→图表→PDF自动化 | reportlab |

### 3. 可视化报表（/reports/）
| 图表 | 内容 | 来源 |
|:---|:---|:---|
| 2025产品生命周期矩阵.png | 销售额vs增长率散点图 | PowerBI |
| 2025产品销售趋势.png | 月度销售额趋势 | PowerBI |
| 产品预警表.png | 连续下滑产品清单（标红） | PowerBI |
| Top10产品.png | 销售额排名 | PowerBI |
| 月度销售额.png | 基础趋势图 | PowerBI |

| 多产品趋势对比图.png | Python生成的双产品对比 | matplotlib |

### 4. PowerBI交互看板（/power BI/）
- **销售预警看板.pbix**：动态筛选+条件格式标红+趋势追踪
- **曙光电子销售及产品分析.pbix**：基础分析版本

## 技术栈
- **MySQL 8.0**：窗口函数（LAG/LEAD/OVER）、复杂子查询
- **Python**：pandas数据处理、matplotlib可视化、reportlab PDF生成、sqlalchemy数据库连接
- **PowerBI Desktop**：交互可视化、DAX基础、条件格式

## 数据范围
- **时间跨度**：2024年1月 - 2025年12月（24个月）
- **数据量级**：569条订单记录，100+SKU

## 关键发现
- 16510B点焊支架：2025年多次连续下滑，全年呈衰退趋势
- 面板插脚：2025年12月销售额暴跌93.3%
- 12362支架：2025年8月销售额暴跌96.6%
- 13373B点焊支架：2025年4月峰值后波动下滑
