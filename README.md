# 冲压件厂销售数据分析

## 项目背景
基于制造业业务场景，分析2024-2025年销售数据，优化生产运营决策。

## 数据规模
- 时间跨度：24个月（2024.01-2025.12）
- 订单数量：601条
- 总销售额：962万元

## 技术栈
- Python：Pandas数据清洗、Matplotlib可视化
- SQL：SQLite多维度分析
- Power BI：动态可视化看板

## 核心发现
1. 核心产品（13373B支架）占52%营收（约500万）
2. 2025年销售额同比增长22%（432万→529万）
3. 2月为传统淡季（1.4万），11月峰值66万

## 文件说明
| 文件 | 说明 |
|:---|:---|

| 曙光电子销售及产品分析.pbix | Power BI仪表板（2页） |

## 使用说明
1. 用DBeaver或SQLite工具运行SQL查询
2. 用Power BI Desktop打开.pbix文件查看交互看板
3. 数据已脱敏，不含真实客户信息
