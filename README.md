## 冲压件厂销售数据分析

### 项目背景
基于制造业业务场景，分析 2024-2025 年销售数据，优化生产运营决策。

---

### 数据规模
| 指标 | 数值 |
|:---|:---|
| 时间跨度 | 24 个月（2024.01-2025.12）|
| 订单数量 | 601 条 |
| 总销售额 | 962 万元 |

---

### 技术栈
| 技术 | 用途 |
|:---|:---|
| **Python** | Pandas 数据清洗、Matplotlib 可视化 |
| **MySQL** | 多维度数据查询与分析 |
| **Power BI** | 动态可视化看板 |

---

### 核心发现
- 核心产品（13373B 支架）占 **52%** 营收（约 500 万）
- 2025 年销售额同比增长 **22%**（432 万 → 529 万）
- 2 月为传统淡季（1.4 万），11 月峰值 66 万

---

### 文件说明

| 文件 | 说明 |
|:---|:---|

---

### 使用说明

1. **SQL 查询**
   - 用 **Navicat** 或 **DBeaver** 连接 MySQL 数据库
   - 运行 SQL 查询文件

2. **Power BI 看板**
   - 用 Power BI Desktop 打开 `.pbix` 文件查看交互看板

3. **数据安全**
   - 数据已脱敏，不含真实客户信息
