# 制造业销售数据分析项目

## 项目背景
为中小型冲压件制造企业搭建销售数据分析体系，识别产品趋势与风险，辅助业务决策。

## 核心成果

### 1. SQL分析体系
| 查询文件 | 用途 | 技术点 |
|:---|:---|:---|
| 24-25年度/月度销售额.sql | 基础业绩统计 | GROUP BY聚合 |
| top10产品.sql | 识别核心贡献产品 | ORDER BY+LIMIT |
| 不同产品月度销售额.sql | 产品维度透视 | 多字段分组 |
| 月度销售额环比变化.sql | 月度趋势分析 | 窗口函数LAG |
| 连续2个月销售额下滑预警.sql | 风险产品识别 | 窗口函数嵌套 |
| 24-25产品销售额年增长率.sql | 年度对比分析 | LAG+JOIN |

### 2. 可视化报表（/reports/）
| 图表 | 内容 | 洞察 |
|:---|:---|:---|
| 2025产品生命周期矩阵.png | 销售额vs增长率散点图 | 识别明星/现金牛/潜力/瘦狗产品 |
| 2025产品销售趋势.png | 月度销售额趋势 | 发现主力产品全年波动规律 |

| 月度销售额.png | 基础趋势图 | 淡旺季识别 |

### 3. PowerBI交互看板（/power BI/）
- **销售预警看板.pbix**：动态筛选+条件格式+趋势追踪
- **曙光电子销售及产品分析.pbix**：基础分析版本

## 技术栈
- **MySQL 8.0**：窗口函数（LAG/LEAD）、复杂子查询、CTE
- **Python pandas**：数据清洗（data/清洗后数据2.csv）
- **PowerBI Desktop**：可视化、DAX基础、交互设计

## 数据范围
- **时间跨度**：2024年1月 - 2025年12月（24个月）
- **数据量级**：569条订单记录，100+SKU
- **核心指标**：销售额、销售量、单价趋势、环比增长率、年增长率

## 关键发现
| 发现 | 说明 | 业务建议 |
|:---|:---|:---|
| 主力产品全年衰退 | 某产品2025年多次连续下滑 | 排查客户流失或竞品替代 |
| 年底集中下滑 | 多产品12月销售额骤降 | 关注客户年度预算节奏 |
| 高波动产品 | 部分产品单月涨跌超50% | 建立安全库存机制 |
| 明星产品识别 | 少数产品规模与增长双高 | 优先保障产能与交付 |

## 学习收获
- 掌握MySQL窗口函数实现时序分析（LAG/LEAD/OVER）
- 建立"基础统计→趋势分析→风险预警"的完整分析链路

- 从"描述过去"升级到"预警未来"的数据分析思维
